ЧТО ДАЛЬШЕ:
=====================================================

1. СОЗДАЙТЕ ФАЙЛ .env.local
   - В корне проекта (рядом с package.json)
   - Скопируйте туда содержимое выше

2. ПОЛУЧИТЕ API КЛЮЧИ:

   OpenRouter (для генерации текста):
   → https://openrouter.ai/keys
   → Создайте ключ и пополните баланс на $5-10

   OpenAI (для DALL·E):
   → https://platform.openai.com/api-keys
   → Создайте ключ и добавьте способ оплаты

3. НАСТРОЙТЕ БАЗУ ДАННЫХ:
   
   → https://supabase.com/dashboard/project/osokxlweresllgbclkme/sql
   → Откройте файл supabase/schema.sql из проекта
   → Скопируйте и запустите SQL код

4. СОЗДАЙТЕ STORAGE BUCKETS:
   
   → https://supabase.com/dashboard/project/osokxlweresllgbclkme/storage/buckets
   → Создайте 3 публичных bucket'а:
      - creatives
      - backgrounds
      - renders

5. ЗАПУСТИТЕ ПРОЕКТ:
   
   npm run dev
   
   Откройте: http://localhost:3000
